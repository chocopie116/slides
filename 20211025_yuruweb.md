# タイトル

Firebase, Cloud Run を使ってサービスリリースした話~フルマネージド環境を求めて~

---

# キーメッセージ

フルマネージドな環境最高。インフラ弱者もこれで余裕。

---

# 有益なことはなにか

- フルマネージド最高。
- Firestore まじ便利。KVS なのにクエリが書ける
- 管理画面で集計が必要なら、Firestore -> Bigquery -> Google データポータル
- Cloud Run は設定少なくて良い
